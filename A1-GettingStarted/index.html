Description

We are going to implement a chatting room which allowas user to add frineds, chat with friends, and make public speech sort of things like Slack.

Features

  • Allow user to create account at the very beginning.
  • Allow user to login with existing account.
  • Allow user to log out.
  • Allow user to add friend by user name.
  • Allow user to do private chat with friends.
  • Allow user to create room with room id to do public chat.
  • Allow user to search room with room id then join if willing to.
  • Allow user to upload image in private chat or public chat room.
  • Allow user to send voice message in private chat or public chat room.
  • Allow user to select background music when chatting in private or chat room.

CSS Option

  • Materialize
  • Semantic UI
